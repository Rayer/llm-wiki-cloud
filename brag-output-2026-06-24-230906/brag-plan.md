# Brag Plan: LLM Wiki Cloud

## What is this app?
LLM Wiki Cloud is an ingest-time compile wiki pipeline — it turns raw markdown into interlinked, queryable wiki articles using Gemini, running entirely on GCP Cloud Run. No vector DB. No chunking. No RAG.

## The angle
Everyone is building RAG. LLM Wiki Cloud asks: what if you just… compiled the knowledge at ingest time? The product's quiet confidence is the creative premise — this is infrastructure that doesn't need to shout.

## Hook (first 2-3 seconds)
"Everyone is building RAG."
(pause)
"We didn't."

## Key moments (the middle)
- The RAG problem: chunking, embedding, retrieval, vector DBs — a stack of complexity
- The LLM Wiki answer: ingest-time compile → query-time just read the whole article
- The result: 3 compiled sources, interlinked concepts, zero vector DBs

## Outro / punchline
"LLM Wiki Cloud. Ingest-time compile. Not RAG."
