# PwC Agent Harness Reshapes Retrieval

## Summary
PwC於2026年5月發表論文，比較了grep與vector retrieval在LongMemEval上的表現，並搭配四種Agent Harness。研究發現，grep在處理依賴字面證據的任務上全面優於vector RAG，且Agent Harness的設計差異對最終效能的影響甚至超過檢索方法本身。論文同時點出，Programmatic delivery可能導致「end-to-end brittleness」，並強調Agent時代下需以「retrieval-plus-orchestration」的視角評估檢索系統。
