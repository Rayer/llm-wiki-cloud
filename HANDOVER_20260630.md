# LLM Wiki Cloud — 2026-06-30 Handover

## 總結

修復了 multi-account（rayer/demo/qatester）的 pipeline、index、wikilinks、frontend API 路徑與 CORS。所有改造以「stateless」、「id-based routing」、「case-insensitive wikilinks」為核心設計原則。

## 架構變更

### 1. Stateless Server（移除 Global UserID/ProjectID）
- 移除 `cfg.UserID`、`cfg.ProjectID`、`cfg.DefaultUserID` from config struct
- 移除 `defaultRequestScope` middleware
- GCS/Firestore client 改為 bucket-scoped，per-request 透過 `h.gcs.WithScope()` 動態 scoping
- **JWT_SECRET 必須設為 Cloud Run env var**（Dockerfile 不包含 config.toml）

### 2. MVP Handler 清除
- 移除 `internal/handler/handler.go`、`raw.go`、`metrics.go`
- 移除所有 `/api/` route（保留 `/api/v1/`）
- 前端 API_URL 改為 Cloud Run direct URL（繞過 Cloudflare CORS preflight redirect）

### 3. Route 改名
- `/api/v1/sources/:slug` → `/api/v1/sources/:id`
- `/api/v1/concepts/:slug` → `/api/v1/concepts/:id`
- Handler 內 `c.Param("slug")` → `c.Param("id")`

### 4. Pipeline Worker 重構
- gsutil rsync → GCS FUSE symlinks
- gcloud auth → metadata server token
- Secret Manager gcloud CLI → REST API (urllib)
- `.olw/` SQLite → local `/tmp/.olw`（copy-in/copy-out，gcsfuse 不支援 random write）
- Pipeline 完成後自動 rebuild-index via BFF API

### 5. ID Map 與 Rebuild-Index
- Files without frontmatter `id` → auto-generate MD5(content)[:12]
- Rebuild-index handler: X-User-ID/X-Project-ID header 優先（非 context）
- Rebuild-index endpoint 移至 JWTAuth 之外（pipeline worker 直接呼叫）
- `GET /api/v1/index` 回傳完整 id_map

### 6. Wikilinks Rewrite
- `[[Concept Name]]` → `[name](/concepts/{id})`（標準 Markdown）
- `[[Source Name]]` → `[name](/sources/{id})`
- Case-insensitive matching（`Yoyu` ↔ `YoYu`）
- Unmatched wikilinks → 保持原文 `[[text]]`（不產生 broken link）

### 7. Canonical URL
- Redirect `/sources/{id}-{slug}` → `/sources/{id}`（僅用 ID）
- `{id}`-only URL 不觸發 redirect（`slug==""` canonical）

### 8. Frontend 修復
- API_URL → Cloud Run direct URL（bypass Cloudflare）
- ListClient links → `/{type}/{id}`（不再包含 slug）
- Internal wikilinks 不開 `target="_blank"`
- ListClient 空結果不 cache，error 清 cache

### 9. Swagger
- 參數 `slug` → `id`
- 新增 `GET /api/v1/index` endpoint
- BearerAuth security scheme

### 10. Deploy 優化
- `.gcloudignore` + `.dockerignore`：archive 1.9GB → 820KB
- Cloud Build time: 6min → 3min
